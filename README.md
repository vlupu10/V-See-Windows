# V-See Windows

Windows-specific repository for the V-See photo viewer application.

## About

This repository contains Windows-specific builds, configurations, and documentation for V-See, a high-performance desktop photo viewer written in Python using PyQt6.

## Development Workflow

- **Main branch**: Protected - no direct commits allowed
- **Develop branch**: All development work happens here
- Create feature branches from `develop` and merge back via pull requests

## Getting Started

See the main [V-See repository](https://github.com/vlupu10/V-See) for the full project documentation and source code.
